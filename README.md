### Prequisites

* [Node.js](https://nodejs.org/) v12

### Install Dependencies 

```
npm i
```

### Compile JavaScript

```
npm run build
```

### Run Tests

```
npm test
```

### Run Coverage

```
npm run cov
```

### View Coverage

```
npm run viewcov
```
